Create a Graphics Window

  • Load Pygame model

    Pygame is a cross-platform set of Python modules designed for writing video games. It includes computer graphics and sound libraries designed to be used with the Python programming language.

    import pygame
from pygame import *

  • Initialization and exit of the game

    Method: Description
    pygame.init() initialize all imported pygame modules, use it before another modules.
    pygame.quit() uninitialize all pygame modules, use it when you want exit the game. 
    def main():
    pygame.init()

if __name__ == '__main__':
    main()
    pygame.quit()

  • Create the main game window

    pygame has a special method pygame.display for creating and managing game windows

    Method Description
    pygame.display.set_mode() Initialize the game display window
    pygame.display.update() Refresh the screen content display 
        set_mode(size=(0, 0), flags=0, depth=0, display=0, vsync=0) -> Surface

    This function will create a display Surface. The arguments passed in are requests for a display type. The actual created display will be the best possible match supported by the system.

    Parameter

    • The size argument is a pair of numbers representing the width and height.
    • The flags argument is a collection of additional options.
    • The depth argument represents the number of bits to use for color.

    Here are the display flags you will want to choose from:

    pygame.FULLSCREEN    create a fullscreen display
pygame.DOUBLEBUF     recommended for HWSURFACE or OPENGL
pygame.HWSURFACE     hardware accelerated, only in FULLSCREEN
pygame.OPENGL        create an OpenGL-renderable display
pygame.RESIZABLE     display window should be sizeable
pygame.NOFRAME       display window will have no border or controls
pygame.SCALED        resolution depends on desktop size and scale graphics
pygame.SHOWN         window is opened in visible mode (default)
pygame.HIDDEN        window is opened in hidden mode

    def main():
    pygame.init()
    # Set the windows
    screen = pygame.display.set_mode(size=(480, 700), flags=0)

if __name__ == '__main__':
    main()
    pygame.quit()

  • Game loop

    • In order to ensure that the game program does not exit immediately after it is started, a game loop is usually added to the game program
    • The so-called game loop is an infinite loop
    • Below the code for creating the game window, add an infinite loop
    def main():
    pygame.init()
    # Set the windows
    screen = pygame.display.set_mode(size=(480, 700), flags=0)
    while True:
        pygame.display.update()

  • Add Events Listener

    When we run the program, we find that the window has appeared, but we cannot manipulate the window, or even close the window. Therefore, we should add an Events Listener to control the closing of the window.

    # Set Events Listener
for events in pygame.event.get():
    if events.type == pygame.QUIT:
        pygame.quit()
        exit()

Draw image

In the game, most of the game elements that can be seen are images

  • We need to follow three steps to display the images in windows:

    • Use pygame.image.load() to load image data
    • Call the blit method to draw the image to the specified position
    • Call the pygame.display.update() method to update the display of the entire screen

  • Background and Player

    def main():
    pygame.init()
    # Set the windows
    screen = pygame.display.set_mode(size=(480, 700), flags=0)
    background = pygame.image.load("images/background.png")
    player = pygame.image.load("images/me1.png")
    while True:
        # Set Events Listener
        for events in pygame.event.get():
            if events.type == pygame.QUIT:
                pygame.quit()
                exit()
        screen.blit(background, (0, 0))
        screen.blit(player, ((480 - 102) / 2, 500))
        pygame.display.update()
    • NOTE: Please blit the Background first, then blit the Player so that the Player is on top of the Background
